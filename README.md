## Development

### Pre-required

- [mise](https://mise.jdx.dev)

### Setup

1. Trust project directory and install tools.

    ```sh
    mise trust -q && mise install
    ```

2. Setup tools.

    ```sh
    mise setup
    ```

### Fix and Check

1. Fix.

    ```sh
    mise fix
    ```

2. Check.

    ```sh
    mise check
    ```

### Sync template updates

```sh
copier update
```
